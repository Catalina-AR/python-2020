@media queries

CSS uses the @media to detect information about the device being used to access the website. For example, it can be used to detect the size of the device, its capabilities, and its orientation. For this course, we will be using it to detect the device size, but keep ind mind that is capable of much more

Desktop: Width greater than or equal to 1024px

Tablet: Width 481px and 1023px

Phone: Width less tan or equal to 480 px
